## Scope Tecnico

### 1. Root Layout (`src/app/layout.tsx`)

```tsx
import { DM_Serif_Display, DM_Sans, Space_Mono } from 'next/font/google'

// Fontes com subsets e display swap
const dmSerifDisplay = DM_Serif_Display({
  weight: '400',
  subsets: ['latin'],
  variable: '--font-serif',
  display: 'swap'
})

const dmSans = DM_Sans({
  subsets: ['latin'],
  variable: '--font-sans',
  display: 'swap'
})

const spaceMono = Space_Mono({
  weight: ['400', '700'],
  subsets: ['latin'],
  variable: '--font-mono',
  display: 'swap'
})
```

- Metadata base: `title: "Instituto VI3"`, `description`, `lang: "pt-BR"`
- Classes das fontes no `<html>` via CSS variables
- Importar `Nav` e `Footer` como children wrapper

### 2. Nav (`src/components/organisms/Nav.tsx`)

**Desktop (>900px):**
- Posicao: fixed, top: 0, full width
- Background: `rgba(10, 22, 40, 0.85)` + `backdrop-filter: blur(16px) saturate(180%)`
- Border bottom: `rgba(30, 159, 212, 0.2)` 1px
- Logo: SVG placeholder (texto "VI3" estilizado por ora)
- Links: Sobre, Fundadores, Ecossistema, Contato — font-family mono, uppercase, letter-spacing 0.04em
- CTA: Button "Ecossistema" com bg cyan, hover brightness

**Mobile (<900px):**
- Logo + hamburguer icon (3 linhas)
- Click: menu slide-down ou overlay com links stacked
- Animacao suave (300ms ease)
- Fechar ao clicar em link ou fora

**Acessibilidade:**
- `<nav>` com `aria-label="Navegacao principal"`
- Hamburguer: `aria-expanded`, `aria-controls`
- Links: `aria-current="page"` na pagina ativa
- Focus trap no menu mobile quando aberto

### 3. Footer (`src/components/organisms/Footer.tsx`)

- Background: navy-mid (#0D1F3C)
- Border top: line color `rgba(30, 159, 212, 0.2)`
- Layout: 3 colunas desktop, stack mobile
  - Col 1: Logo + descricao curta
  - Col 2: Links legais (Termos de Uso, Politica de Privacidade) — placeholder hrefs
  - Col 3: Redes sociais (YouTube, Instagram, TikTok) — icones SVG ou placeholders
- Copyright: "2026 Instituto VI3. Todos os direitos reservados."
- Font: DM Sans, body-sm size

## Riscos

| Risco | Probabilidade | Impacto | Mitigacao |
|-------|---------------|---------|-----------|
| backdrop-filter nao suportado em mobile antigo | Baixa | Nav sem glass effect | Fallback: background solido navy-backdrop |
| FOUT com 3 fontes Google simultaneas | Media | Flash visual no carregamento | next/font com display: swap + preload das fontes criticas |
| Noise texture SVG impacta performance mobile | Baixa | FPS drop | Opacity minima (3%), desabilitar em prefers-reduced-motion |
| Hamburguer menu sem focus trap | Media | Falha a11y | Implementar focus trap com useEffect + keydown listener |
