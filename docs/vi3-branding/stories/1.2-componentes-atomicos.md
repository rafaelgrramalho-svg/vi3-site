## Scope Tecnico

### 1. Button (`src/components/atoms/Button.tsx`)

```tsx
interface ButtonProps {
  variant: 'primary' | 'ghost'
  size?: 'default' | 'sm'
  href?: string        // Se fornecido, renderiza como <a>
  children: React.ReactNode
  onClick?: () => void
  className?: string
}
```

**Primary:**
- Background: cyan (#1E9FD4)
- Text: white (#F0F4FF)
- Font: DM Sans, font-weight 500, letter-spacing 0.06em, uppercase
- Padding: `0.75rem 1.75rem` (default), `0.5rem 1.25rem` (sm)
- Hover: brightness(1.1), translateY(-1px)
- Border-radius: 0 (ZERO — cantos retos)

**Ghost:**
- Background: transparent
- Border: 1px solid `rgba(30, 159, 212, 0.3)`
- Text: cyan (#1E9FD4)
- Hover: background `rgba(30, 159, 212, 0.08)`, translateY(-1px)

**Transicao:** `transition: all 200ms cubic-bezier(0.16, 1, 0.3, 1)`

### 2. Tag (`src/components/atoms/Tag.tsx`)

```tsx
interface TagProps {
  children: React.ReactNode
  className?: string
}
```

- Font: Space Mono, uppercase
- Size: label-section (0.68rem)
- Letter-spacing: 0.14em
- Color: cyan (#1E9FD4)
- Sem background, sem border (apenas texto estilizado)

### 3. Badge (`src/components/atoms/Badge.tsx`)

```tsx
interface BadgeProps {
  status: 'ativo' | 'em-producao' | 'em-breve'
  children?: React.ReactNode
}
```

| Status | Cor | Label Default |
|--------|-----|---------------|
| ativo | state-green (#22C55E) | Ativo |
| em-producao | state-amber (#F59E0B) | Em producao |
| em-breve | state-purple (#A855F7) | Em breve |

- Font: Space Mono, label-sm (0.65rem), uppercase
- Dot indicator: 6px circulo da cor correspondente
- Layout: dot + texto inline

### 4. Card (`src/components/molecules/Card.tsx`)

```tsx
interface CardProps {
  children: React.ReactNode
  className?: string
  hover?: boolean  // default true
}
```

- Background: navy-mid (#0D1F3C) ou transparent com border
- Border: 1px solid `rgba(30, 159, 212, 0.2)`
- Border-radius: 0 (ZERO)
- Padding: `1.75rem`
- Hover (se habilitado):
  - `transform: translateY(-2px)`
  - `box-shadow: 0 0 20px rgba(30, 159, 212, 0.08)`
  - `border-color: rgba(30, 159, 212, 0.4)`
- Transicao: 200ms cubic-bezier(0.16, 1, 0.3, 1)

### 5. ProductCard (`src/components/molecules/ProductCard.tsx`)

```tsx
interface ProductCardProps {
  name: string
  pilar: string
  description: string
  status: 'ativo' | 'em-producao' | 'em-breve'
  type: 'curso' | 'mentoria' | 'ferramenta' | 'livro'
  ctaText: string
  ctaHref: string
}
```

**Badges de tipo (cor do PRD secao 3.3):**
| Tipo | Cor |
|------|-----|
| Curso | cyan (#1E9FD4) |
| Mentoria | gold (#F59E0B) |
| Ferramenta/SaaS | green (#22C55E) |
| Livro/Info-produto | amber (#F59E0B) |

**Layout interno:**
1. Type badge (topo, cor do tipo) + Status badge (ao lado)
2. Nome do produto (h3, DM Serif Display)
3. Pilar (tag pequena)
4. Descricao curta (body-sm, white-dim)
5. CTA button (ghost variant, com href externo)

### 6. FounderCard (`src/components/molecules/FounderCard.tsx`)

```tsx
interface FounderCardProps {
  name: string
  role: string
  bio: string
  imageSrc?: string   // placeholder se nao fornecido
  credentials?: string[]
}
```

**Layout:**
1. Foto circular (placeholder: iniciais em bg cyan-glow) — 120px
2. Nome (h3, DM Serif Display)
3. Role (Tag atom, cyan)
4. Bio (body-sm, white-dim, max 3 linhas)
5. Credentials (lista se fornecidas)

**Nota:** Foto da Stephany ainda nao disponivel — usar placeholder elegante

## Riscos

| Risco | Probabilidade | Impacto | Mitigacao |
|-------|---------------|---------|-----------|
| FounderCard sem foto real da Stephany | Alta | Card incompleto | Placeholder elegante com iniciais em bg cyan-glow |
| Tipos de produto podem mudar (adicionar novos) | Baixa | Props desatualizadas | TypeScript union type extensivel, facil de adicionar |
| Hover effects pesados em mobile touch | Media | UX estranha | Desabilitar translateY em touch devices ou usar @media(hover: hover) |
| Mentoria e Livro com mesma cor (amber/gold) | Baixa | Confusao visual | Diferenciar: Mentoria=gold(#F59E0B), Livro=amber mais escuro ou usar icone diferenciador |
